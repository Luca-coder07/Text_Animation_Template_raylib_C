# TEXT ANIMATION WITH RAYLIB IN C

## Compile and Run:<br>
- In Linux:
  > gcc <file> -o main -lraylib -lm -lGl -lpthread -ldl -lrt -lx11
  > ./main
- In Windows:
  > gcc <file> -o main -lraylib -lgdi32 -lwinmm
  > ./main
## Pre-requists
- math.h
- string.h
- Raylib (a framework for Game Development in C)
